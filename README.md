## Install

Using pip:

```bash
$ pip3 install gadalang
```

## Documentation

Build the doc with:

```bash
$ make html
```

You can find the latest documentation on [gadalang.readthedocs.io](https://gadalang.readthedocs.io/).

## Testing

The `test` directory contains many tests that you can run with:

```python
$ tox .
```
