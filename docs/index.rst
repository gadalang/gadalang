Installation
------------

.. code-block:: bash

   $ pip3 install gadalang


Testing
-------

The ``test`` directory contains many tests that you can run with:

.. code-block:: bash

   $ tox .


Dependencies

- Python 3.7+
- gada_
- gada-pyrunner_
- pygada-runtime_
- gadalang-lang_
